# ProjectStorageBox 📦

Sistema de gerenciamento de projetos pessoais de programação, 
desenvolvido em Java com foco em boas práticas de 
Orientação a Objetos.

## 🚀 Sobre o projeto

O ProjectStorageBox permite registrar e organizar projetos 
de código, controlando informações como nome, linguagem, 
descrição e status de cada projeto.

Desenvolvido como projeto de portfólio durante meus estudos 
de Java e Orientação a Objetos.

## 🛠️ Tecnologias utilizadas

- Java
- Orientação a Objetos
- Eclipse IDE

## ✅ Funcionalidades

- Cadastro de projetos com nome, linguagem e status
- Validação defensiva de dados obrigatórios
- Controle de status: Não Iniciado, Em Andamento e Concluído
- Tratamento de erros com IllegalArgumentException
